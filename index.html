Tentang

Halo, Saya adalah Alan Hertanto Guspermana, Saya merupakan seorang Freelance Programmer yang terbiasa bekerja dibawah tekanan, bagi saya kepuasan client merupakan kepuasan pribadi saya. Saya dapat dikatakan terbiasa menggunakan Unity Engine 5-2019 untuk berbagai macam keperluan, seperti Game Developing, Application Developing, dll. Saya juga terbiasa menggunakan Framework CodeIgniter, Google Sketchup, CorelDraw. Saya juga selalu tertarik akan hal baru.
